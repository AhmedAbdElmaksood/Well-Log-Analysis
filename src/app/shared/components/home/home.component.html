Well Log data analysis is an open source web application to help geologists, geophysicists and any geoscience student to analysis wells files data and make some decisions based on it.

Our Vision

Our objective is to make well log data analysis easy for everyone. Doesn’t need to pay too much for a software and go through a long process for installation. Our application will be online, easy to access from any device at any time without too much complexity to use. Displaying some important graphs and make some important calculations on the data.

We analyze your data with good visuals

With few clicks you can easily analyze the geologic formations penetrated by a drill hole. If the hole has been drilled by using coring techniques, the core provides a visual record of the formations and rock types encountered. The description (log) of the core provides the basic data used in geologic analysis, interpretation, and resource calculations.
